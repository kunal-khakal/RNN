# 🎬 RNN Movie Review Sentiment Analysis

## 📌 Project Overview

This project uses a **Recurrent Neural Network (RNN)** to classify movie reviews as **Positive 👍 or Negative 👎** using the IMDB dataset.

---

## 📂 Files

* `08) Codes.ipynb` → Model training & testing
* `02) IMDB Dataset.csv` → Dataset (via Git LFS)

---

## 🧠 Technologies Used

* Python 🐍
* TensorFlow / Keras
* NumPy & Pandas
* Jupyter Notebook

---

## ⚙️ How to Run

1. Install dependencies:

```bash
pip install numpy pandas tensorflow
```

2. Open notebook:

```bash
jupyter notebook
```

3. Run all cells

* Predicts sentiment of reviews
